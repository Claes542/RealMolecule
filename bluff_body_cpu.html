Compressible Navier–Stokes bluff-body flow as a RealThermo CPU companion to bluff_body_mach2.html. Same physics, 1D-indexed Float32Arrays, ~280 lines.

Equations: ∂ρ/∂t + ∇·m = 0, ∂m/∂t + ∇·(m u) + ∇p = ∇·(ν∇u) + f, ∂e/∂t + ∇·(e u) + p∇·u = ν|∇u|² with p = γe and u = m/ρ. Bluff body = penalty force f = −K u on the yellow rectangle. Dirichlet inflow at left (constant u0, ρ = 1, e = 1), Neumann elsewhere.

Blue line: ρ−1 cross-cut. Green line: ux cross-cut. Red dots: particle traces (reset cadence via slider). Yellow: bluff-body region.

Dcum = ∫∫ ν|∇u|² dx dt — cumulative viscous dissipation, the 2nd-Law witness for this flow.
